# ТЗ: Titanium Dawn — Premium AI Music Player

## 1. Цель

Доработать существующий музыкальный веб-плеер до уровня **портфолио-проекта вайб-кодера**.

Проект должен выглядеть как законченная презентационная страница авторского музыкального релиза, созданного с использованием нейросетей.

---

## 2. Исходное состояние

Уже реализовано:

- [x] Воспроизведение музыкального трека (`music.mp3`)
- [x] Кнопки Play и Pause
- [x] Прогресс-бар с перемоткой
- [x] Переключение светлой и тёмной темы (с сохранением в `localStorage`)
- [x] Отображение обложки (`cover.jpeg`)
- [x] Управление громкостью и перемотка ±10 сек
- [x] Отображение текущего времени и длительности трека
- [x] Адаптивная вёрстка (базовая)
- [x] Публикация проекта в интернете

> **Примечание:** в текущем коде нет отдельной кнопки Stop — есть Play/Pause. Если Stop требуется по ТЗ заказчика, добавить без нарушения существующей логики Play/Pause.

## 4. Функциональные требования

### 4.1. Премиальный дизайн

- [ ] Атмосферный дизайн в стиле современного музыкального стримингового сервиса
- [ ] Внешний вид законченного продукта для портфолио, не учебного примера
- [ ] Единая визуальная система для заголовков, подписей и элементов управления
- [ ] Улучшенная читаемость и визуальная иерархия

### 4.2. Анимации интерфейса

- [ ] Плавное появление элементов при загрузке страницы
- [ ] Современные hover-эффекты на интерактивных элементах

- [ ] Мягкое свечение вокруг обложки
- [ ] Лёгкая анимация обложки
- [ ] Без перегруженности эффектами — аккуратный внешний вид

### 4.4. Прогресс-бар

- [ ] Плавная анимация заполнения
- [ ] Современный визуальный дизайн
- [ ] Удобное отображение текущей позиции трека

### 4.5. Отображение времени

- [ ] Показывать текущее время воспроизведения
- [ ] Показывать общую длительность трека
- [ ] Формат: `00:00 / 03:45` (или эквивалент с разделителем `/`)

### 4.6. Адаптивность

- [ ] Корректное отображение на мобильных устройствах
